# React Pokedex

**React Pokedex** is a project made with learning purposes.

[Demo](https://pokedex.magarcia.now.sh)

## Getting started

```sh
git clone git@github.com:magarcia/react-pokedex.git
cd react-pokedex
yarn install
yarn start # open localhost:3000
```
