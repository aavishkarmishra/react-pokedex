# React Pokédex

**React Pokédex** is a project made with learning purposes.

[Demo](https://pokedex.magarcia.now.sh)

## Getting started

```sh
git clone git@github.com:magarcia/react-pokedex.git
cd react-pokedex
npm install
npm start # open localhost:3000
```
